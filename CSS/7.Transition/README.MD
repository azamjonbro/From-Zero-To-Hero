## ğŸ§© Advanced Toggle Example (Bounce effect)

``` css
.toggle {
  width: 50px;
  height: 26px;
  background: #ccc;
  border-radius: 20px;
  position: relative;
  cursor: pointer;
  transition: background .3s ease;
}
.toggle::after {
  content: "";
  width: 22px;
  height: 22px;
  background: white;
  border-radius: 50%;
  position: absolute;
  left: 2px;
  top: 2px;
  transition: transform .25s cubic-bezier(.34,1.56,.64,1);
}
.toggle.active {
  background: #4ade80;
}
.toggle.active::after {
  transform: translateX(24px);
}
```
