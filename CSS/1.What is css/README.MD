## üì¶ CSS Box Model nima?

Har bir HTML elementi aslida **kvadrat (quti)** ko‚Äòrinishida bo‚Äòladi va 4 qismdan iborat:

```
+---------------------------+
|        margin             |
|  +---------------------+  |
|  |       border        |  |
|  |  +---------------+  |  |
|  |  |    padding     | |  |
|  |  | +-----------+ | |  |
|  |  | |  content  | | |  |
|  |  | +-----------+ | |  |
|  |  +---------------+  |  |
|  +---------------------+  |
+---------------------------+
```

### **1) Content**

Matn yoki rasm joylashadigan asosiy maydon.

### **2) Padding**

Content atrofidagi ichki bo‚Äòsh joy.

```css
padding: 20px;
```

### **3) Border**

Elementning chegarasi.

```css
border: 2px solid black;
```

### **4) Margin**

Element tashqi tomondagi bo‚Äòsh joy.

```css
margin: 20px;
```

## Box Model hisoblanishi

```css
.box {
  width: 200px;
  height: 100px;
  padding: 20px;
  border: 5px solid red;
  margin: 30px;
}
```

**Real kenglik hisoblanishi:**

```
width + padding + border
200 + 20 + 20 + 5 + 5 = 250px
```

## `box-sizing` xulqi

### **content-box (default)**

`width` faqat contentga tegishli.

### **border-box (tavsiya etiladi)**

`width` ichiga padding va border ham kiradi.

```css
* {
  box-sizing: border-box;
}
```

Natija: element o‚Äòlchami **aniq** bo‚Äòladi.
